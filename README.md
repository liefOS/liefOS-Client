# liefOS

This is the official client for liefOS.

## Features

* Auto-updates liefOS: When a new liefOS update was released, The client will install the new release as soon as possible. Keeping your liefOS patched!
<small>Just to be clear, liefOS and liefOS Client are two completely different things. liefOS Client displays and runs liefOS and updates it. liefOS Client are still manually updated by the user by installing a new release from the website. A warning will be shown in liefOS Client when there's a new client version!</small>
